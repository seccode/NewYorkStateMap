About the Map

This interactive map of New York State was created with the mission of providing an easy-to-use tool for learning about the State of New York. Using this map, you will be able to explore areas of New York such as the Adirondacks, New York City, and the Finger Lake Region.

The features included in this map have been broken down into 9 categories:

  • Boundary Lines
  • Political Regions
  • Economic Regions
  • Demographics
  • Tax Parcels/Property
  • Agriculture
  • Environment
  • Animals
  • Infrastructure

Using open-source data provided by New York State, the USDA, and other government agencies, we are able to create an accurate and up-to-date map of millions of features across the State.
